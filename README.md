# 疫情居家办公管理系统 java267

## 项目概述

疫情居家办公管理系统是一款基于Springboot+Vue的前后端分离项目，采用Javaee技术构建，实现了MVC三层架构。系统利用Spring、SpringBoot、Mybatis、Vue和Maven技术栈进行开发，提供了一套简洁、易操作的Web界面。

## 技术栈

- **后端**: Springboot、Javaee、Mybatis
- **前端**: Vue
- **构建工具**: Maven
- **数据库**: MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户管理
- 办公任务管理
- 信息发布
- 数据统计分析
- 系统设置

## 系统角色

- 系统管理员
- 普通用户
- 高级用户

## 运行环境

- **后端**: 
  - Java 1.8 或以上版本
  - Springboot 2.x
- **前端**:
  - Vue 2.x
- **数据库**: 
  - MySQL 5.7 或以上版本

## 部署步骤

1. 准备运行环境（Java、MySQL等）
2. 克隆项目代码
3. 构建后端（使用Maven）
4. 部署前端资源
5. 导入数据库
6. 配置系统参数
7. 启动后端服务
8. 访问前端界面
